# 更新日志

## [1.3.0] - 2024-01-XX

### 新增功能
- 🌐 国际化工具 (i18nUtils) - 支持数字、货币、日期格式化
- 🔐 加密工具 (cryptoUtils) - 哈希、随机字符串、UUID、Base64编码
- 🎬 动画工具 (animationUtils) - 淡入淡出、滑动、缩放等动画效果
- 📍 地理位置工具 (geoUtils) - 位置获取、距离计算、方位角计算
- 📝 表单处理工具 (formUtils) - 表单序列化、验证、自动保存
